# Snitap

Landing page desenvolvida com HTML5 e CSS3 para apresentar a **Snitap**, uma marca fictícia voltada à mobilidade urbana sobre patins. O projeto tem como foco a criação de interfaces modernas, animações em CSS e layouts responsivos.

## Sobre o projeto

O **Snitap** é uma landing page criada para divulgar um serviço relacionado à mobilidade urbana e ao uso de patins como meio de transporte e lazer. A interface foi desenvolvida com ênfase na experiência do usuário, utilizando uma identidade visual moderna e componentes bem organizados.

Durante o desenvolvimento foram aplicados conceitos de HTML semântico, Flexbox, animações com CSS e técnicas de responsividade para diferentes tamanhos de tela.

## Como executar o projeto

1. Clone este repositório:

```bash
git clone https://github.com/daniellydev/snitap.git
```

2. Acesse a pasta do projeto:

```bash
cd snitap
```

3. Abra o arquivo `index.html` em seu navegador.
